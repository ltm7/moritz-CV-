LTM Webseite für ein Online Shop von mechanischen Bauteilen
KURZE BESCHREIBUNG

Diese Webseite ist eine dynamische Webseite für online Shop, das nur als Beispiel benutzen werden kann. Diese Webseite, dass ich dynamisch programmiert habe, ist für ein Shop , die seine Kunden oder Patnerunternehmen mechanischen Bauteilen verkauft. Dafür musste die Webseite zuerst mit CSS und HTML statisch programmiert werden und danach anhand von PHP und MySQL dynnamisch umgestellt werden. Um diese Webseite herstellen und optiemieren zu können, hatte ich einen lokalen Server benötigt, der auf meinen Computer installiert war.Dafür habe XAMPP Control Panel installiert und ein paar Parameter geändert. Hier können Sie ein Übersicht dieser Webseite finden:Mechashop Moritz.Falls Sie diese nicht öffnen können,können Sie eine Übersicht einsehen.

Ich hätte diese dynamische Webseite direkt aus der Quelldatei verwalten können, aber es war sinnvoller, meine dynamische Website mit einem Datenbank(Verwaltungsystem) zu verwalten. Mit diesem System wäre ich nicht verpflichtet, meine Quelldatei zu verwenden, und jeder Benutzer ohne Programmierkenntnissen könnte diese Website leicht personalisieren. Hier können Sie ein Übersicht die Webseite meines Verwaltungsystems finden:Verwaltungssystem Mechashop.Falls Sie diese nicht öffnen können,können Sie eine Übersicht einsehen.
